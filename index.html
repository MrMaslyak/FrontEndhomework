• git status - показує поточні зміни в коді
  • git add - додає всі файли до майбутнього коміту
  • git remote - дозволяє створювати, переглядати та видаляти підключення до інших репозиторій
  • git checkout - дозволяє переміщатися між гілками, створеними командою git branch
  • git push - використовується для публікації локальних змін, що вивантажуються, в центральному репозиторії.
